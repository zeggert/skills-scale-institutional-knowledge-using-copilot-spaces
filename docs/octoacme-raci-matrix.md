# OctoAcme RACI Matrix

This document provides a RACI (Responsible, Accountable, Consulted, Informed) matrix for key project activities. Use this matrix to clarify ownership and communication expectations across roles.

---

## RACI Legend

| Code | Meaning |
|------|---------|
| **R** | Responsible — Does the work to complete the task |
| **A** | Accountable — Ultimately answerable for the task's completion |
| **C** | Consulted — Provides input or expertise before a decision is made |
| **I** | Informed — Kept up-to-date on progress or decisions |

## How to Use This Matrix

1. **Before starting a new activity**, review the matrix to confirm who is accountable and responsible.
2. **Escalate conflicts** if multiple people believe they are accountable for the same task — clarify with the Project Manager.
3. **Adjust for your team**: This matrix is a template. Modify assignments based on your team structure and project needs.
4. **Review periodically**: Update the matrix as roles or processes evolve.
